# Simple Budget Tracker

A minimalist, production-ready budget tracking application built with Next.js 14, Prisma, NextAuth.js, and Tailwind CSS.

## ✨ Features

-   **User Authentication:** Secure sign-up, login, and session management.
-   **Transaction Management:** Create, read, update, and delete income and expense transactions.
-   **Dashboard:** At-a-glance summary of total income, expenses, and current balance for the month.
-   **Responsive Design:** Fully usable on desktop and mobile devices.
-   **Type-Safe:** Built with TypeScript from end to end.

## 🛠️ Tech Stack

-   **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **ORM:** [Prisma](https://www.prisma.io/)
-   **Database:** [PostgreSQL](https://www.postgresql.org/)
-   **Authentication:** [NextAuth.js](https://next-auth.js.org/)
-   **Validation:** [Zod](https://zod.dev/)
-   **UI Components:** Custom components inspired by [shadcn/ui](https://ui.shadcn.com/).
-   **Testing:** [Jest](https://jestjs.io/) & [Playwright](https://playwright.dev/)
-   **Containerization:** [Docker](https://www.docker.com/)
